# 모델 vs 에이전트 차이

## 한 줄 요약
- **모델**: 글 받아 글 뱉는 뇌.
- **에이전트**: 그 뇌가 손발(도구) 쓰며 스스로 일하는 존재.

## 모델 (Model, LLM)
- 입력: 텍스트 → 출력: 텍스트.
- 다음 단어를 예측할 뿐.
- 기억 없음. 매번 새로 시작.
- 직접 행동 못 함 (파일 못 읽고, 인터넷 못 봄).
- 예: GPT, Claude "그 자체".

## 에이전트 (Agent)
- 모델 + 도구 + 루프.
- 목표 받으면 스스로 단계 나눠 실행.
- 도구로 행동: 파일 읽기, 검색, 명령 실행.
- 결과 보고 다음 행동 결정 (반복).
- 기억/상태 유지 가능.
- 예: Claude Code, Cursor 에이전트.

## 핵심 차이 표

| 구분 | 모델 | 에이전트 |
|------|------|----------|
| 본질 | 예측기 | 모델 + 실행 틀 |
| 행동 | 못 함 | 도구로 함 |
| 반복 | 한 번 답 | 루프 돌며 일함 |
| 기억 | 없음 | 있음 (파일/DB) |
| 목표 | 답변 | 작업 완수 |

## 비유
- 모델 = 똑똑한 사람, 손발 묶이고 눈 가린 채 질문에만 답.
- 에이전트 = 같은 사람에게 손발 풀고 도구함 주고 "이거 끝내" 시킴.
